# LotteryPlate
-
###这个转盘效果在项目的实际使用的基础上扩展而来.现在将它开源出来,方便小伙伴们使用.
####特点:
1.稳定.目前经过了真实项目 [镭达钱包](http://sj.qq.com/myapp/detail.htm?apkName=com.igeek.radarwallet) 中4天6000+的点抽奖实践,未发现bug;  
2.支持多种属性,配置成多个item的抽奖转盘.属性请参考文后.
